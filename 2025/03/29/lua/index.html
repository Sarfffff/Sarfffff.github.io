3. 搜索路径与模块加载

require 函数会依据 package.path 来查找模块文件。你可以通过修改 package.path 来调整搜索路径。

4. 重复加载与缓存

require 函数会对已经加载的模块进行缓存，多次调用 require 加载同一个模块时，实际上只会加载一次。若要重新加载模块，可以清除缓存。

5. 加载 C 模块

除了 Lua 模块，require 函数还能加载 C 模块。C 模块通常是编译好的动态链接库（Windows 下为 .dll，Linux 下为 .so）。
